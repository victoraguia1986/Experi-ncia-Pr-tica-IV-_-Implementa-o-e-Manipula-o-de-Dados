# Sistema de Gerenciamento de Cemitério – Banco de Dados  
Projeto académico desenvolvido para as Experiências Práticas 1 e 2 da disciplina de Modelagem de Banco de Dados.

---

## Objetivo do Projeto
Criar, modelar e implementar a base de dados de um sistema de gestão de cemitério, incluindo:

- Registo de falecidos  
- Gestão de sepulturas e jazigos  
- Concessões e renovação  
- Serviços e ordens de manutenção  
- Documentos associados
